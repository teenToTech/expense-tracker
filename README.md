# 💰 Expense Tracker (Python CLI Project)

A simple command-line based Expense Tracker built using Python.  
It helps users track their daily expenses and stores data permanently using a JSON file.

---

## 🚀 Features

- Add new expenses with category, amount, and description
- View all expenses
- Delete expenses by index
- Calculate total expenses
- Data stored permanently in `expenses.json`

---

## 🛠️ Tech Stack

- Python
- JSON (for data storage)
- File Handling

---

## 📁 Project Structure


Expense Tracker/
│
├── expense_tracker.py
└── expenses.json (auto-created)

## ▶️ How to Run

1. Make sure Python is installed

2. Run the program:
```bash
python expense_tracker.py
Follow menu options:
1. Add expenses  
2. Show expenses  
3. Delete expenses  
4. Total expenses  
5. Exit
